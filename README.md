# MySSH - 远程服务器运维工具

一个类似 XShell 的现代化远程服务器运维工具，使用 Vue 3 + Tauri 构建。

![MySSH 界面截图](home.png)

## 功能特性

### 核心功能

- 🖥️ **多服务器管理** - 支持同时管理多个服务器连接，服务器列表支持收起/展开
- 💻 **终端管理** - 功能强大的终端，支持主题、字体、分屏、录制/回放
- 📁 **文件管理器** - 可视化文件管理，支持上传下载、重命名、权限管理、预览、对比、批量操作、搜索
- 📊 **系统监控** - 实时监控 CPU、内存、磁盘、网络，支持历史图表、告警阈值、进程详情、网络连接、磁盘分区
- 📝 **日志查看器** - 实时日志查看，支持过滤、搜索、多文件查看、导出
- 🤖 **AI 助手** - 智能助手，帮助处理服务器运维任务
- 📦 **传输管理** - 文件传输进度显示、断点续传、传输队列管理、传输历史记录

### 界面特性

- 🎨 **现代化界面** - 深色主题，类似 VS Code 的设计风格
- 📐 **灵活布局** - 可调整的面板大小，支持收起/展开
  - 服务器列表：左侧，支持收起/展开
  - 终端：上方，可调整高度
  - 文件管理：下方，支持收起/展开
  - AI助手/监控：右侧，交替显示，支持收起/展开
- ⚡ **高性能** - 基于 Tauri，原生性能
- 💾 **状态持久化** - 自动保存布局设置、告警阈值等配置

### 终端功能

- 🎨 **多主题配色** - 支持多种终端主题（深色、浅色、高对比度等）
- 📏 **字体调节** - 可自定义字体大小和字体族
- ⇄ **分屏功能** - 支持水平和垂直分屏，同时查看多个终端
- ⏺️ **录制/回放** - 录制终端会话，支持回放查看
- 🧹 **清屏/重连** - 快速清屏和重新连接功能

### 文件管理功能

- 📤 **文件上传/下载** - 支持文件上传和下载，带进度显示
- ✏️ **重命名** - 支持文件和文件夹重命名
- 🔒 **权限管理** - 支持 chmod 权限编辑（八进制和复选框方式）
- 👁️ **文件预览** - 支持文本和图片文件预览
- 🔍 **文件对比** - 支持两个文件的对比查看，高亮差异
- ☑️ **批量操作** - 支持多选文件进行批量操作（Ctrl/Cmd + 点击，Shift + 点击）
- 🔎 **搜索功能** - 支持按文件名搜索

### 系统监控功能

- 📈 **实时监控** - CPU、内存、磁盘、网络实时监控
- 📊 **历史图表** - CPU 和内存使用率历史趋势图表
- ⚠️ **告警阈值** - 可设置 CPU、内存、磁盘告警阈值
- 🔍 **进程详情** - 查看系统进程列表和详细信息
- 🌐 **网络连接** - 查看网络连接状态和详细信息
- 💾 **磁盘分区** - 查看磁盘分区使用情况

### 传输管理功能

- 📊 **进度显示** - 实时显示文件传输进度
- 🔄 **断点续传** - 支持大文件断点续传
- 📋 **队列管理** - 管理文件传输队列
- 📜 **历史记录** - 查看文件传输历史记录

### 日志查看功能

- 📝 **实时查看** - 实时查看日志文件
- 🔍 **过滤/搜索** - 支持日志过滤和搜索
- 📄 **多文件查看** - 同时查看多个日志文件
- 💾 **导出功能** - 支持日志导出

## 技术栈

- **前端框架**: Vue 3
- **状态管理**: Pinia
- **路由**: Vue Router
- **构建工具**: Vite
- **桌面框架**: Tauri
- **后端**: Rust (待实现)

## 使用说明

### 添加服务器

1. 点击左侧服务器列表顶部的 `+` 按钮
2. 填写服务器信息：
   - 服务器名称（可选）
   - 主机地址（必填）
   - 端口（默认 22）
   - 用户名（必填）
   - 密码或密钥路径

### 连接服务器

1. 在服务器列表中点击服务器项
2. 点击 `▶` 按钮连接服务器
3. 连接成功后会自动打开终端标签页

### 界面布局

- **服务器列表**：左侧显示，点击 `◀` 按钮可收起，收起后显示 `🖥️` 按钮可展开
- **终端**：位于上方，可通过拖拽中间的分割器调整高度
- **文件管理**：位于下方，点击 `▼` 按钮可收起，收起后显示 `📁` 按钮可展开
- **AI助手/监控**：位于右侧，点击右侧按钮栏的 `🤖` 或 `📊` 按钮可展开，两者交替显示

### 使用终端

- 在终端中输入命令并回车执行
- 支持命令历史记录（上下箭头键）
- **主题设置**：点击 `⚙️` 按钮，选择主题配色
- **字体调节**：在设置面板中调整字体大小和字体族
- **分屏功能**：点击 `⇄` 或 `⇅` 按钮进行水平或垂直分屏
- **录制/回放**：点击 `⏺️` 按钮开始录制，点击 `▶️` 按钮回放
- 点击"清屏"按钮清除终端输出
- 点击"重连"按钮重新连接服务器

### 使用文件管理器

1. 文件管理器位于终端下方
2. **基本操作**：
   - 双击文件夹进入目录
   - 双击文件打开预览
   - 选择文件后点击"下载"按钮下载文件
   - 点击"上传"按钮上传本地文件
   - 点击"新建"按钮创建文件夹
   - 选择文件后点击"删除"按钮删除文件
3. **高级功能**：
   - **重命名**：选择文件后点击"重命名"按钮，或右键菜单选择"重命名"
   - **权限管理**：选择文件后点击"权限"按钮，可设置文件权限（chmod）
   - **文件预览**：双击文件或右键菜单选择"预览"，支持文本和图片预览
   - **文件对比**：右键菜单选择"对比"，选择两个文件进行对比
   - **批量操作**：按住 `Ctrl/Cmd` 键点击文件进行多选，或按住 `Shift` 键进行范围选择
   - **搜索**：在搜索框中输入文件名进行搜索

### 使用系统监控

1. 点击右侧按钮栏的 `📊` 按钮打开系统监控
2. **监控卡片**：垂直排列显示 CPU、内存、磁盘、网络监控信息
3. **历史图表**：点击 `📊 历史` 按钮查看 CPU 和内存使用率历史趋势
4. **告警设置**：点击 `⚙️ 告警` 按钮设置 CPU、内存、磁盘告警阈值
5. **进程详情**：点击 `🔍 进程` 按钮查看系统进程列表
6. **网络连接**：点击 `🌐 网络` 按钮查看网络连接状态
7. **磁盘分区**：点击 `💾 磁盘` 按钮查看磁盘分区使用情况

### 使用日志查看器

1. 在文件管理器中找到日志文件
2. 双击日志文件或右键菜单选择"查看日志"
3. **功能**：
   - 实时查看日志内容
   - 使用过滤功能过滤日志
   - 使用搜索功能搜索日志内容
   - 同时查看多个日志文件
   - 导出日志文件

### 使用传输管理器

1. 文件上传/下载时会自动显示传输进度
2. 点击传输管理器查看传输队列和历史记录
3. **功能**：
   - 查看传输进度
   - 暂停/继续传输（支持断点续传）
   - 管理传输队列
   - 查看传输历史记录

### 使用 AI 助手

1. 点击右侧按钮栏的 `🤖` 按钮打开 AI 助手
2. 在对话框中输入问题或命令
3. AI 助手会帮助处理服务器运维任务
